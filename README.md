# 📊 E-Commerce Sales Analysis Dashboard

## Overview

This project presents an interactive Power BI dashboard built to analyze e-commerce sales data. The dashboard helps monitor sales performance, profit, customer behavior, product categories, payment modes, and monthly trends.

---

## Dashboard Preview

![Dashboard](Images/Ecommerce%20Dashboard.png)

---

## Features

- Interactive KPI Cards
- Sales Analysis
- Profit Analysis
- Customer Analysis
- Category-wise Analysis
- Payment Mode Analysis
- Monthly Profit Trend
- State Filter
- Quarter Slicer
- Drill-down Analysis

---

## Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Data Modeling
- CSV Dataset

---

## Dataset

- Orders.csv
- Details.csv

---

## KPIs

- Total Sales
- Total Profit
- Total Quantity Sold
- Average Order Value (AOV)

---

## Project Learnings

- Created an interactive Power BI dashboard for e-commerce sales analysis.
- Used slicers and filters to enable dynamic user interaction.
- Built data relationships and performed data modeling.
- Developed DAX calculations for business metrics.
- Designed multiple visualizations to uncover actionable insights.

---

## Key Insights

- Clothing contributes the highest quantity sold.
- COD is the most preferred payment mode.
- Profit varies across different months.
- Printers generate the highest profit among sub-categories.
